# Práctica 2 - Modelo Estrella GameZoneDW

Proyecto académico de un Data Warehouse para la tienda ficticia GameZone.

## Base de datos
- GameZoneDW
- Modelo: Esquema estrella

## Tablas de dimensiones
- DimFecha
- DimProducto
- DimCliente
- DimEmpleado
- DimSucursal

## Tabla de hechos
- FactVentas

## Análisis realizados
- Productos más vendidos
- Ventas por sucursal
- Ventas por empleado
- Ventas por categoría

## Tecnologías utilizadas
- SQL Server Management Studio
- SQL Server
- GitHub
